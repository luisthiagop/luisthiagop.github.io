Ajuda

Para os campos de posições usar apenas valores inteiros positivos;

Para o campo de orientação inicial usar apenas valores N,S,L,O;

Para a lista de canteiros usar apenas valores inteiros separados por virgula e qubra de linha;

Após o cálculo é possível ver o trajeto do robô clicando em desenhar. Essa função funciona melhor com hortas quadradas com dimensões maior 3.
